Для решения данного ТЗ я решил использовать django  в комбинаций с django rest что бы продемонстрировать умение в написание REST API view в app users отправляет запрос по юрл http://localhost:8000/api/v1/users/ а вот view в rest_api обрабатывает этот запрос и дает ответ

Также написал написал dockerfile который создает image нашего веб приложения и написал docker compose в который поднимает postgres в случае если бы мы хотели использовать postgres вместо sqlite (но для этот ноужно будет дополнительно установить зависимость psycopg2 и поменять db в settings.py)
